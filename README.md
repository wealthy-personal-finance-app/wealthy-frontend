# Wealthy Frontend 🚀

A high-fidelity frontend foundation for **Wealthy**, a modern personal finance and cash flow management application. 

This repository contains the interactive UI, authentication flows, and data visualization dashboards designed to give users a comprehensive view of their financial ecosystem.

## 🛠️ Tech Stack
* **Framework:** React + Vite
* **Styling:** Tailwind CSS
* **Routing:** React Router DOM
* **Visualizations:** Recharts / Nivo (Sankey Diagrams)

## ✨ Key Features
* **Authentication:** Secure Sign In and Sign Up split-screen flows with robust form state management.
* **Interactive Cash Flow:** A progressive-disclosure Sankey diagram mapping income sources to expense buckets.
* **Transaction Management:** Dynamic "Add Transaction" modals and "Autopilot" scheduling drawers.
* **Responsive Design:** Pixel-perfect implementation of Figma design tokens for a premium, dark-mode fintech experience.
